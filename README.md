# Elaf Group — Commercial Performance Dashboard

## Overview
A Power BI dashboard analyzing the commercial performance 
of Elaf Group's 9-hotel portfolio across Saudi Arabia.

## What's Inside
- KPI Overview: Revenue, RevPAR, Occupancy, NPS
- Pareto Analysis: Core revenue-driving hotels
- Commercial Deep Dive: ADR vs RevPAR benchmarking
- Seasonal Analysis: Hajj, Umrah & Regular seasons
- NPS & Quality: Linking satisfaction to revenue

## Key Insight
Hajj season generates RevPAR of SAR 1,021 — 
yet NPS drops to 51. A strategic opportunity 
to invest in service quality during peak season.

## Tools Used
- Power BI Desktop
- Microsoft Excel
- DAX Measures

## Note
All data is simulated for demonstration purposes.
Methodology applies to actual hotel operational data.
